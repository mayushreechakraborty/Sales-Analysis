# 📊 E-commerce Sales Dashboard (Power BI)

## 📌 Project Overview
This project presents an interactive **E-commerce Sales Dashboard** built using **Power BI**.  
The dashboard provides insights into sales performance, customer behavior, and product trends to support business decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Identify top-selling products  
- Understand customer purchasing patterns  
- Track monthly and yearly trends  
- Visualize key business metrics  

---

## 🛠️ Tool Used
- Power BI 📊  

---

## 📂 Dataset
- File: `ecommerce_sales_data_1.xlsx`  
- Includes:
  - Order details  
  - Customer information  
  - Product categories  
  - Sales and profit data  

---

## 📊 Dashboard Features

### 🔹 Key Metrics
- Total Sales  
- Total Profit  
- Number of Orders  
- Customer Count  

### 🔹 Visualizations
- 📈 Sales Trend (Monthly/Yearly)  
- 📊 Category-wise Sales  
- 🏆 Top Products  
- 👥 Top Customers  
- 🌍 Region-wise Sales
